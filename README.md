# Windows Backup Script

Add this tutorial to my repo "Windows_tutorials"

## Cloning

1. Open Git Bash
1. Run backup script by executing

        git clone 
        cd windows_backup
        
## Usage

        ./backup

## Script description

Script operations

1. Format backup drive
1. Backup folders
1. Show message that the backup has completed

## Task scheduling

- with opened powershell window
- as current user
